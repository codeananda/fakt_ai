## Installation

1. Install [Poetry](https://python-poetry.org/)
    ```
    curl -sSL https://install.python-poetry.org | python3 -
    ```

2. cd into the project folder and install dependencies with
    ```
    $ poetry install
    ```

3. Activate the virtual env
    ```
    $ poetry shell
    ```
